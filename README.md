## 🚀 How to Run

**No installation needed!**

```bash
# Clone the repository
git clone https://github.com/ZoyInfinity-Hospitality-Private-Limited/milan-infotech-website.git

# Open in browser
cd milan-infotech-website
open index.html        # macOS
start index.html       # Windows
```

Or simply **double-click `index.html`** in your file explorer.

## 📬 Contact Form Setup (Formspree)

To make the contact form send real emails to your inbox:

1. Go to [formspree.io](https://formspree.io) and sign up free
2. Create a new form → enter `marketing@zoypg.com` as the email
3. Copy your Form ID (e.g. `xpzgkwqr`)
4. Open `contact.html` → find `YOUR_FORM_ID` → replace it with your ID

```html
<!-- Find this line in contact.html -->
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">

<!-- Replace with your actual ID -->
<form action="https://formspree.io/f/xpzgkwqr" method="POST">
```
